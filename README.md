# bigbrother
To run this code u need to install nodejs and git.
Then :
```sh
git clone https://github.com/elyas74/bigbrother
cd bigbrother
npm install
```
now for developing u can run :
```sh
nodemon
```
it's recommended to use [nodemon](https://www.npmjs.com/package/nodemon) fo development and [pm2](https://www.npmjs.com/package/pm2) for public release.
